First Install bootstrap using CLI command :

```bash
npm install bootstrap
```

Next add these in `angular.json` :

```bash
"scripts": [
    "./node_modules/bootstrap/dist/js/bootstrap.bundle.min.js"
],
"styles": [
    "./node_modules/bootstrap/dist/css/bootstrap.min.css"
],
```
